Drawing with Drones
a web app created with P5.js
The idea for my project came from a drones class I took at Parsons the New School for Design. I visited and photographed using a drone at a summer camp in Westchester County, New York. For this project, I was inspired by the historical home that was on the property. Using P5, Node.js, Express, and Sockets I was able to create this multi-player web application.
