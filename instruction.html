An Instruction Set for Strangers
The idea for my project came from a visit to Washington Square Park, New York. After observing the space, with the help of a group we created multiple iterations for a design intervention that people could interact with. By using user testing in the actual space, we were able to determine if we accomplished our goal of creating interactions.
